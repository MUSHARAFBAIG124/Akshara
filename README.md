# Akshara

Akshara-Deepa Tutor is an Android app for SSLC self-study. It includes subject progress, chapter completion, textbook access, audio summaries, quizzes, and strength tracking.

## Build

```powershell
.\gradlew.bat :app:assembleDebug
```

The debug APK is generated at:

```text
app/build/outputs/apk/debug/app-debug.apk
```
